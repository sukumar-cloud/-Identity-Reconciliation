# Full-Stack Application

A modern full-stack web application with React frontend and Node.js backend.

## Project Structure

- **Backend**: Node.js with TypeScript, REST API
- **Frontend**: React with TypeScript, Vite, Tailwind CSS
- **Deployment**: Render for backend

## Setup Instructions

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features

- Identity verification service
- Contact management
- Activity tracking
- Dashboard with analytics
- Responsive design

## Tech Stack

**Backend:**
- Node.js
- TypeScript
- Express
- PostgreSQL

**Frontend:**
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
